# Материалы, запрещённые к публичной раздаче

Файлы в этой папке не должны переноситься обратно в `public` без письменной
проверки юристом и подтверждающих документов.

- `reel4-patient-testimonial.*` — узнаваемое лицо пациента, сведения о здоровье,
  благодарность и история лечения.
- `reel6-specific-case.*` — конкретный случай диагностики.

Для повторной публикации недостаточно заменить имя на «Пациент»: нужно отдельно
проверить ограничения медицинской рекламы, врачебную тайну, согласие на
распространение персональных данных и права на все элементы ролика.
